Términos de Uso

Última actualización: 4 de noviembre de 2025

Bienvenido a Alfred. Estos términos de uso rigen tu acceso y uso del software Alfred. Al descargar, instalar o usar Alfred, aceptas estos términos.

1. Aceptación de los Términos

Al acceder y utilizar Alfred ("la Aplicación", "el Software"), aceptas estar legalmente vinculado por estos Términos de Uso. Si no estás de acuerdo con estos términos, no descargues, instales ni uses la Aplicación.

2. Descripción del Servicio

Alfred es un asistente de escritorio con inteligencia artificial que:

  • Procesa documentos personales localmente en tu dispositivo
  • Utiliza modelos de IA ejecutados completamente offline (mediante Ollama)
  • Proporciona búsqueda inteligente y respuestas basadas en tus documentos
  • Almacena todos los datos exclusivamente en tu dispositivo local
  • No requiere conexión a internet para su funcionamiento normal

Alfred se distribuye bajo Apache License 2.0. Esta licencia te otorga:

  • Derecho a usar el software comercial y no comercialmente
  • Derecho a modificar el código fuente
  • Derecho a distribuir copias del software
  • Derecho a distribuir versiones modificadas
  • Concesión de licencia de patente de contribuyentes

El texto completo de la licencia está disponible en: Apache License 2.0

3.2 Código Abierto

El código fuente completo de Alfred está disponible públicamente en: GitHub - Alfred

4. Requisitos del Sistema

Para usar Alfred necesitas:

  • Sistema Operativo: Windows 10 o superior (64-bit)
  • RAM: Mínimo 8 GB (16 GB recomendado)
  • Almacenamiento: 20 GB libres (30 GB recomendado)
  • Software Adicional: Ollama instalado para ejecutar modelos de IA
  • GPU (Opcional): NVIDIA, AMD o Apple Silicon para mejor rendimiento

5. Uso Aceptable

5.1 Usos Permitidos

Puedes usar Alfred para:

  • Procesar y analizar tus documentos personales
  • Obtener respuestas basadas en tu información local
  • Gestionar tu información personal de forma privada
  • Desarrollo, modificación y mejora del software
  • Uso comercial dentro de tu organización

5.2 Usos Prohibidos

NO debes usar Alfred para:

  • Procesar información ilegal o que infrinja derechos de terceros
  • Crear versiones modificadas que violen la licencia Apache 2.0
  • Distribuir malware o código malicioso bajo el nombre "Alfred"
  • Realizar ingeniería inversa con fines maliciosos
  • Cualquier actividad ilegal o no autorizada

6. Privacidad y Datos Locales

Alfred está diseñado con privacidad como prioridad:

  • No recopilamos ni transmitimos datos personales
  • Todo el procesamiento ocurre localmente en tu dispositivo
  • No hay servidores externos ni telemetría
  • Tus datos permanecen bajo tu exclusivo control

Para más detalles, consulta nuestra Política de Privacidad.

7. Propiedad Intelectual

7.1 Software Alfred

Alfred es software de código abierto bajo Apache License 2.0. Los derechos de autor pertenecen a los contribuyentes originales.

7.2 Tus Datos

Tú eres el único propietario de:

  • Todos los documentos que procesas con Alfred
  • Todas las conversaciones y consultas generadas
  • Toda la información almacenada localmente

7.3 Marcas Comerciales

"Alfred" y el logotipo asociado son marcas del proyecto. El uso de estas marcas en versiones modificadas debe cumplir con la licencia Apache 2.0.

8. Dependencias y Software de Terceros

Alfred utiliza software de terceros:

  • Ollama: Servidor de modelos LLM local (licencia MIT)
  • Electron: Framework para aplicaciones de escritorio (licencia MIT)
  • FastAPI: Framework Python para APIs (licencia MIT)
  • LangChain: Framework de LLM (licencia MIT)
  • ChromaDB: Base de datos vectorial (licencia Apache 2.0)
  • PyTorch: Framework de deep learning (licencia BSD)

Cada componente está sujeto a sus propias licencias respectivas.

9. Modelos de IA

Los modelos de IA utilizados por Alfred:

  • Se descargan a través de Ollama desde repositorios públicos
  • Están sujetos a sus propias licencias individuales
  • Se ejecutan completamente en tu dispositivo local
  • No envían ni reciben datos de servidores externos

Es tu responsabilidad verificar las licencias de los modelos que descargues y uses.
