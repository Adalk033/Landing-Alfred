Características

Potencia e Inteligencia en tu Escritorio

Todo lo que necesitas en un asistente con IA, sin comprometer tu privacidad

IA 100% Local

Todo el procesamiento se ejecuta en tu dispositivo usando Ollama. Sin conexiones externas, sin envío de datos.

Privacidad Total

Encriptación AES-256-GCM para tus datos. Nunca enviamos información a servidores externos. Tu privacidad es nuestra prioridad.

RAG Inteligente

Búsqueda vectorial con ChromaDB y generación de respuestas contextuales basadas en tus documentos personales.

Aceleración GPU

Soporte automático para NVIDIA CUDA, AMD ROCm y Apple Silicon MPS para un rendimiento óptimo.

Interfaz Moderna

Chat con efectos visuales, renderizado Markdown completo, tablas profesionales y gestión de conversaciones.

Múltiples Formatos

Procesa PDF, Word, Excel, texto plano y más. Gestiona grandes colecciones de documentos sin esfuerzo.

Privacidad

Tus Datos, Solo Tuyos

A diferencia de otros asistentes con IA, Alfred no requiere conexión a internet para funcionar. Todo se procesa localmente en tu dispositivo.

  • Sin conexiones a APIs externas ni servicios en la nube
  • Encriptación AES-256-GCM para datos sensibles
  • Almacenamiento local seguro de documentos y conversaciones
  • Código abierto y auditable bajo licencia MIT

FAQ

Preguntas Frecuentes

¿Es realmente gratis?

Sí, Alfred es completamente gratis y de código abierto bajo licencia MIT. No hay costos ocultos, suscripciones ni límites de uso.

¿Necesito conexión a internet?

Solo para la descarga inicial de modelos de IA. Una vez instalado, Alfred funciona completamente offline.

¿Qué tan privado es realmente?

100% privado. Todo se procesa localmente. No hay telemetría, no hay servidores externos, no se envían datos a ningún lugar.

¿Puedo usar mi propia GPU?

Sí, Alfred detecta automáticamente tu GPU NVIDIA, AMD o Apple Silicon y la utiliza para acelerar el procesamiento.

¿Qué formatos de documentos soporta?

PDF, Word (.docx), Excel (.xlsx), PowerPoint (.pptx), archivos de texto (.txt, .md) y más.

¿Está disponible para Mac o Linux?

De momento solo para Windows. Las versiones para macOS y Linux están en desarrollo.
